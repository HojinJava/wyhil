## 제출 규칙

작업 완료 후 모든 커밋을 1개로 squash하고 위에서 checkout한 모델 브랜치를 대상으로 PR을 생성한다.
PR 본문에 `References \#{ISSUE_NUMBER}` 를 포함하고 본인 모델 라벨을 추가한다.

PR 생성 전 Commits 탭을 확인한다. 본인 커밋이 1개만 보이면 정상이다. 여러 개라면 main에서 브랜치를 딴 것이므로 PR을 닫고 위 명령어부터 다시 시작한다.
